## Frequently Asked Questions

**EN:** What is MyVendorHub?

**HE:** מה זה MyVendorHub?

**EN:** MyVendorHub is a vendor order management platform for online stores. It automatically notifies suppliers when their products are ordered, tracks Accept / Reject / Other responses, sends reminders, and gives you a dashboard of vendor performance — so you spend less time on manual emails and WhatsApp threads.

**HE:** MyVendorHub היא פלטפורמה לניהול הזמנות ספקים לחנויות אונליין. היא מודיעה אוטומטית לספקים כשמזמינים את המוצרים שלהם, עוקבת אחרי תגובות אישור / דחייה / אחר, שולחת תזכורות, ומציגה דשבורד של ביצועי ספקים — כדי שתשקיעו פחות זמן באימיילים ידניים ובשרשורי וואטסאפ.

**EN:** Do I need a MyVendorHub account?

**HE:** האם נדרש חשבון MyVendorHub?

**EN:** Yes. Without a connection the plugin shows “Not connected” and does not transmit order data. A free Starter plan is available after you connect.

**HE:** כן. בלי חיבור התוסף מציג \"Not connected\" ואינו מעביר נתוני הזמנות. תוכנית Starter חינמית זמינה אחרי החיבור.

**EN:** Where do I manage vendors and settings?

**HE:** איפה מנהלים ספקים והגדרות?

**EN:** In the MyVendorHub web dashboard at https://www.myvendorhub.com. This plugin connects your WooCommerce store; day-to-day vendor management happens in MyVendorHub. After connect, use Open MyVendorHub from the plugin settings for one-click return (no re-login).

**HE:** בלוח הבקרה של MyVendorHub בכתובת https://www.myvendorhub.com. התוסף מחבר את חנות ה־WooCommerce; ניהול הספקים השוטף מתבצע ב־MyVendorHub. אחרי החיבור השתמשו ב־Open MyVendorHub מהגדרות התוסף לחזרה בלחיצה אחת (בלי התחברות מחדש).

**EN:** Are WooCommerce REST API keys required?

**HE:** האם נדרשים מפתחות REST API של WooCommerce?

**EN:** No. Order forwarding works through this plugin alone. Optional WooCommerce REST keys can be added later in MyVendorHub for richer dashboard reads and advanced features.

**HE:** לא. העברת הזמנות עובדת דרך התוסף בלבד. מפתחות REST אופציונליים של WooCommerce אפשר להוסיף אחר כך ב־MyVendorHub לקריאות דשבורד עשירות יותר ולתכונות מתקדמות.

**EN:** How do vendors get notified?

**HE:** איך הספקים מקבלים התראות?

**EN:** When an order includes a vendor’s products, MyVendorHub emails them (and optionally WhatsApp on eligible plans) with order details and one-click Accept / Reject / Other buttons. You configure contacts and notification channels in the Vendors list inside MyVendorHub.

**HE:** כשהזמנה כוללת מוצרים של ספק, MyVendorHub שולח להם אימייל (ואופציונלית וואטסאפ בתוכניות מתאימות) עם פרטי ההזמנה וכפתורי אישור / דחייה / אחר בלחיצה אחת. מגדירים אנשי קשר וערוצי התראה ברשימת הספקים בתוך MyVendorHub.

**EN:** What if a vendor doesn’t respond?

**HE:** מה קורה אם ספק לא מגיב?

**EN:** Enable smart reminders in MyVendorHub Settings. You can schedule up to three follow-ups and optionally restrict them to working hours so vendors aren’t contacted at night.

**HE:** הפעילו תזכורות חכמות בהגדרות MyVendorHub. אפשר לתזמן עד שלושה מעקבים ולהגביל אותם לשעות עבודה כדי שלא יישלחו הודעות בלילה.

**EN:** Does this work with Dokan, WC Vendors, or similar plugins?

**HE:** האם זה עובד עם Dokan, WC Vendors או תוספים דומים?

**EN:** Vendor names are read from common product / line-item meta keys (_vendor, vendor, _wcv_vendor, _dokan_vendor_id). You can also set a custom vendor meta key in MyVendorHub Settings.

**HE:** שמות ספקים נקראים ממפתחות מטה נפוצים של מוצר / פריט שורה (_vendor, vendor, _wcv_vendor, _dokan_vendor_id). אפשר גם להגדיר מפתח מטה מותאם אישית בהגדרות MyVendorHub.

**EN:** Does the plugin store MyVendorHub passwords?

**EN:** Order and fulfillment data needed for vendors to ship — including customer name, shipping address, and phone when present on the order. MyVendorHub does not sell personal data. See https://www.myvendorhub.com/privacy.

**HE:** נתוני הזמנה ומימוש הנדרשים לספקים למשלוח — כולל שם לקוח, כתובת משלוח וטלפון כשהם קיימים בהזמנה. MyVendorHub אינה מוכרת נתונים אישיים. ראו https://www.myvendorhub.com/privacy.

**EN:** Is it compatible with HPOS?

**HE:** האם יש תאימות ל־HPOS?

**EN:** Yes. The plugin declares compatibility with WooCommerce High-Performance Order Storage.

**HE:** כן. התוסף מצהיר על תאימות לאחסון הזמנות בעל ביצועים גבוהים של WooCommerce.

בסביבות MyVendorHub מקומיות / עצמיות, הגדירו את `VENDORHUB_WC_CONNECT_SECRET` ב־`wp-config.php` כדי להפעיל כפתור **Direct connect (dev)** לרישום HMAC. לא בשימוש ב־SaaS הציבורי.

= מה זה MyVendorHub? =

MyVendorHub היא פלטפורמה לניהול הזמנות ספקים לחנויות אונליין. היא מודיעה אוטומטית לספקים כשמזמינים את המוצרים שלהם, עוקבת אחרי תגובות אישור / דחייה / אחר, שולחת תזכורות, ומציגה דשבורד של ביצועי ספקים — כדי שתשקיעו פחות זמן באימיילים ידניים ובשרשורי וואטסאפ.

כן. בלי חיבור התוסף מציג "Not connected" ואינו מעביר נתוני הזמנות. תוכנית Starter חינמית זמינה אחרי החיבור.

לא. העברת הזמנות עובדת דרך התוסף בלבד. מפתחות REST אופציונליים של WooCommerce אפשר להוסיף אחר כך ב־MyVendorHub לקריאות דשבורד עשירות יותר ולתכונות מתקדמות.

= איך הספקים מקבלים התראות? =

כשהזמנה כוללת מוצרים של ספק, MyVendorHub שולח להם אימייל (ואופציונלית וואטסאפ בתוכניות מתאימות) עם פרטי ההזמנה וכפתורי אישור / דחייה / אחר בלחיצה אחת. מגדירים אנשי קשר וערוצי התראה ברשימת הספקים בתוך MyVendorHub.
